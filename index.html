Application Overview

This application will take existing quizzes and make them into a fun game, like kahoot.
It will allow user input and allow the user to make their own quizzes and test on themselves.

Feature List:

  • home page
  • take quiz
  • create a quiz
  • average out all of the scores
  • contact page

Future Feature List:

  • kahoot version of quiz

Basic Technical Features:

  • decide whether or not user inputed the correct answer
  • average out all the scores of the quizzes

Advanced Technical Features:

  • allow user to set up an account
  • allow user to set up a variety of different questions
